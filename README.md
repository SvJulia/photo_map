Прототип системы для отображения фотографий из Instagramm.

Используется Instagramm API и Google Maps API. 
Пока работает на локальной машине.

Чтобы получить фотографии нужно построить многоугольник на карте, кликая по ней. 
После того, как многоугольник построен, надо замкнуть его, кликнув по маркеру. 
Затем пройдёт некоторое время и начнут появляться фотографии в углах многоугольника.

Пока системы выдаёт 500 фотографий, но можно изменить это значение, изменив параметр maxPhotosCount на желаемый.
